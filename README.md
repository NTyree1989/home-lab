- Windows system troubleshooting
- Operating system recovery techniques

## Projects

### Windows Upgrade and Recovery
Upgraded a personal workstation from Windows 10 to Windows 11 and practiced system recovery methods using DiskPart and external imaging tools.

### Home Network and IoT Environment
Designed and maintain a 2.4 GHz home network supporting IoT devices including smart lighting and audio systems.

### Console OS Troubleshooting
Diagnosing Xbox boot failure caused by corrupted operating system partitions and testing recovery techniques including external reimaging and NVMe adapter workflows.

### Command Line Disk Management
Used DiskPart and command-line utilities to partition external drives and prepare system recovery media.
